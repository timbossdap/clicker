🎮 GAME FEATURES:

  Center: Big Click Button (280×280px)
    → Generates score based on multipliers
    → Click to earn!

  Left Panel: Upgrade Buttons
    → 4 tiers of multiplier upgrades
    → Reset button (clear everything)
    → Rebirth button (prestige mechanic)
    → Gacha Shop button (spend tokens)

  Right Panel: Gacha Shop (when open)
    → Common Pack   (1 token)  → +0.1x
    → Rare Pack     (2 tokens) → +0.3x
    → Epic Pack     (3 tokens) → +0.8x
    → Legendary     (5 tokens) → +2.0x

  Top: Stats Display
    → Current score (formatted: K/M/B/T)
    → Active multipliers (only > 1.0x shown)
    → Rebirth status
